# 软件工程

## 前端

* 目录：Vue_ViewUI

* 使用方法：

```terminal
npm install
npm run init
npm run dev
```
### 目录树
* src：源文件
    * styles：
        * common.css：没啥用的全局样式
    * views：vue视图组件
      * adminView
        * admin.vue：管理员端
      * clientView：
        * client.vue：客户端
        * clientLogin.vue：客户端登录
      * receptionView：
        * reception.vue：前台端
      * connect.vue：网络连接组件
      * welcome.vue：选择登录某端（开发用）
    * app.vue：根组件
    * main.js：主入口
    * router.js：路由配置
    
* babelrc等：配置文件
